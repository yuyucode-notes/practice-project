# practice-project 全栈项目

## 技术栈：

1. 前端：nextjs + antd + charts + pro-antd
2. 后端：nestjs
3. 数据层 ：prisma
4. 记录：sentry
5. 其他


## 项目目录

```
│─┌├┬┼└┴┘│─┌├┬

practice-project
```
